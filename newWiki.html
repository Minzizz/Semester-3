Artikel 2: CSS dan jQuery

CSS mengatur tampilan halaman, seperti warna, jarak, dan posisi elemen. jQuery mempermudah manipulasi DOM dan efek animasi.

Contoh efek: klik tab di atas untuk mengganti artikel!
